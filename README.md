# Cisco Data Science Essentials (Cisco Networking Academy)

This repository contains the coursework, lab solutions, and projects for the Cisco Networking Academy "Data Science Essentials with Python" certification. It is organized by course modules (e.g., Coding for Answers, Data Cleaning, Data Visualization) and includes Python scripts, Jupyter notebooks, Pandas-based analyses, Matplotlib visualizations, and end-to-end lab projects using real-world datasets.

This repository is a learning and reference collection for students and instructors taking the Cisco Networking Academy "Data Science Essentials with Python" course. It gathers module materials, worked examples, lab solutions, and small projects that illustrate core data science workflows: data ingestion, cleaning, exploration, visualization, modeling, and reporting.
