# Skeletal Variation — Data & download instructions

Purpose
- Analyze biological/measurement data (anthropometry): distributions, group comparisons, plots.

Quick links
- Folder on GitHub: [skeletal variation](https://github.com/BhavikBuchke/Cisco-Data-science-program/tree/main/Coding%20for%20answers/skeletal%20variation)
- Data folder: [data](https://github.com/BhavikBuchke/Cisco-Data-science-program/tree/main/Coding%20for%20answers/skeletal%20variation)

Datasets used
- `adult-human-skeleton.csv`
- `bird-neck-bones.csv`
- `mammal-neck-bones.csv`
